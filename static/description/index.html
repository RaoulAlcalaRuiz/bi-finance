Cette application permet de réaliser des objectifs annuels, mensuels et de les comparer à la réalité. La mise en place des objectifs visent divers secteurs de la société :

  • Chiffres d'affaires
  • Satisfaction des clients
  • Gestion des stocks
  • EBIDTA

Comment ça marche ?

L'application est divisée en 4 parties :

  • Les objectifs annuels
  • Les objectifs mensuels
  • les objectifs spécifiques
  • Les rapports

Les objectifs annuels

Cette partie est accessible via le menu "Objectif" > "Objectif annuel". Elle permet de visualiser les objectifs dans la globalité de l'année. Le nom de ces objectifs est directement nommé par l'année.

Les objectifs mensuels

Cette partie est accessible via le menu "Objectif" > "Objectif mensuel". Elle permet de visualiser tous les objectifs mensuels de toutes les années. Bien sûr les objectifs peuvent être regroupé et filtré comme dans n'importe qu'elle application Odoo. Le nom de ces objectifs est directement nommé par le mois et l'année de l'objectif.

Les rapports

Alors pour pouvoir être compris par tout le monde définissons ce qu'est un rapport au sein de cette application. Un rapport est la partie de l'application qui compare les objectifs avec la réalité à l'aide de graphiques. Ces graphiques sont bien sûr téléchargeable et adapté pour l'impression.

Pour acceder au rapport il faut se trouver dans la fenêtre d'un objectif annuel et vous selectionnez l'objectif que vous voulez voir dans le menu d'impression, dans mon cas je vais aller voir le chiffre d'affaires annuel : "print" > "Goal Year Report"

A chaque fois qu'un graphique est affiché, un récapitulatif sous forme de tableau est affiché. Dans le cas de l'objectif chiffres d'affaires : Les tableaux affichent le pourcentage réalisé. La couleur du pourcentage (rouge ou vert) indique si nous somme dans les temps définis par l'objectif. C'est à dire que si nous sommes le 20 Janvier, le résultat du chiffre d'affaires attendu doit être supérieur à 64% car 20 / 31 est égal à 64% (20 = le jour actuel et 31 le nombre de jours dans le mois de janvier)

L'objectif chiffres d'affaires

Pour mettre en place cet objectif il vous faut créer un objectif annuel et un objectif mensuel. Ensuite vous vous dirigez dans le menu "Commerciaux" dans le menu "Chiffres d'affaires".

Comment cela fonctionne ?

Rien de plus simple, il vous suffit d'ajouter des objectifs mensuels pour vos employés en indiquant le chiffre d'affaires qu'ils doivent réaliser par mois grâce aux bons de commandes qu'ils ont fait. Voici un exemple pour être plus concret : Je vais rajouter un objectif à Mitchel Admin et il devra faire un total de 30 000 € en janvier 2020.

Quand vous ajoutez des objectifs aux employés, l'objectif mensuel de Janvier 2020 est passé à 30 000 €, pareil pour l'année 2020. L'objectif annuel reprends tous les objectifs des employés de l'année, pareil pour les objectifs mensuels. Les comparatifs réalité/objectif sont alors visible dans les rapports.

Les rapports disponibles dans cet objectif :

  • Le rapport annuel du chiffres d'affaires de toutes la société
  • Le rapport annuel destiné à l'utilisateur connecté
